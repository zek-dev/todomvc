# Follow the steps mentioned below

1. Ensure to install Maven dependency.
2. Ensure to install "Cucumber Eclipse" and "Natural 0.9" plug in from Eclipse Marketplace.
3. Proceed to import the GitHub project https://github.com/zek-dev/todomvc.git.
4. Switch to main branch.
5. Go to Eclipse > ..eclipse-workspace\ToDoMVCProjectJunit\src\test\java\feature\TestRunner.java > Right click on the file and run as Junit.
